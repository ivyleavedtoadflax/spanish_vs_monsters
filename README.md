# Spanish vs Monsters

A tower defense game for learning Spanish verb conjugations.

## Play Online

The game is deployed at: **https://ivyleavedtoadflax.github.io/spanish_vs_monsters/**

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## How to Play

Answer Spanish verb conjugation questions correctly to build towers that defend against incoming monsters. The game features multiple difficulty levels and verb tenses to help you practice Spanish conjugations.
